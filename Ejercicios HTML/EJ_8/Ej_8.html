Definiciones

  • HTTP: Es el protocolo de transferencia de hypertexto, abreviatura de la forma inglesa "Hypertext Transfer Protocol". Se utiliza en la tranferencia de información en la World Wide Web.
  • HTML: Es el lenguaje utilizado para el desarrollo de páginas web. Sus siglas significan "HyperText Markup Languaje", lo que se traduce como "Lenguaje de Formato de Documentos para Hipertexto".
  • URL: Sus siglas significan "Uniform Resource Locator" y es la dirección específica que se utiliza para que cada uno de los recursos de internet puedan ser identificados y localizados.
  • TCP/IP: Es el protocolo de transmisión de internet, utilizado para permitir la comunicación entre ordenadores pertenecientes a una red. Sus siglas TCP/IP significan "Protocolo de Control de Transmisión/Protocolo de Internet".
  • Internet: Es la conexión que se utiliza a nivel global para la transmisión de información mediante una serie de protocolos denominados TCP/IP.
  • W3C: Es el conjunto de organizaciones internacionales destinadas a establecer una serie de pautas comunes que promuevan la estandarización de los diferentes sistemas de acceso a la web (idioma, navegador, SO, localización geográfica...).
  • Mozilla Foundation: Es una organización dedicada a la creación de software libre. Esta fundación ha creado el navegador conocido como Mozilla Firefox.
